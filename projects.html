My Projects

Student Grading System

For my final project in object oriented programming, I created a student grading system in python. Some of the features include adding studnets to database, recording grades, computing gpa, option to sava and load data and also you could plot charts.

Angry Birds Game

Another project that I contributed to was the spaced theme angry birds game created in unity. I worked on this with a group of other students. The project implements slingshot mechanics, physics‑driven collisions, destructible targets, scoring, and basic level flow. We used C# for gameplay systems (input, projectile behavior, level/state management) and ShaderLab/HLSL to create space‑themed materials and visual effects.

Kart Game

I built a Unity kart racing prototype that shows the basic feel and control. The project implements core driving mechanics (acceleration, braking, steering), a third‑person follow camera, and a simple race loop. Written entirely in C#

Snakes and Ladders Game

I also made a C# version of the classic Snakes and Ladders game. It includes a turn-based loop for multiple players, random dice rolls, and a board with configurable snakes and ladders that move players forward or backward
